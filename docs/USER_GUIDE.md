# User Guide

## 1) Get started
1. Install dependencies:
   - `npm install`
2. Run in development mode:
   - `npm run dev`
3. (Optional) Build desktop app:
   - `npm run build`
   - `npm run tauri build`

## 2) Open a vault
1. Click **Open Folder**.
2. Select the directory containing your markdown files.
3. The app remembers this vault and reopens it on next launch.

## 3) Work with files
- **New file**: click `+ File` (or `Ctrl/Cmd + N`).
- **New folder**: click `+ Folder`.
- **Rename**: select item and press `F2`.
- **Delete**: select item and press `Delete`.

## 4) Edit and save
- Edit markdown in the main editor.
- Save with the app save flow/shortcut.
- Use internal links; backlinks update in the backlinks pane.

## 5) Reference while writing
- Open a PDF from the vault to view it in-app.
- Use **Split View** to keep markdown and PDF side-by-side.
- Open image files for quick preview.

## 6) Search and shortcuts
- Vault search: `Ctrl/Cmd + Shift + F`.
- Toggle sidebar: `Ctrl/Cmd + \`.
- Toggle backlinks: `Ctrl/Cmd + Shift + B`.
- Show shortcuts help: `Ctrl/Cmd + /`.
