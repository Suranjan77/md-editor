# MD Editor V3 User Guide

## Vault Setup
1. Launch MD Editor.
2. Select an existing local folder or create a new one to serve as your vault.
3. The sidebar will index all supported files (.md, .pdf, images).

## Writing and Editing
- **Create**: Use the `New Note` or `New Folder` icons in the sidebar.
- **Markdown**: Full support for headings, bold, italic, lists, wikilinks, code blocks, tables, and math.
- **Formatting**: Use standard shortcuts (`Ctrl+B` for bold, `Ctrl+I` for italic).
- **Search**: `Ctrl+F` to find within the note, `Ctrl+Shift+F` for vault-wide search.

## Reading PDFs
- Click on any PDF to open it natively.
- **Zoom**: `Ctrl+=` to zoom in, `Ctrl+-` to zoom out. `Ctrl+0` to fit width.
- **Highlighting**: Select text and press `Ctrl+H`. Highlights are stored in a sidecar file, preserving the original PDF.
- **Annotations**: View all highlights in the Annotations Panel. You can add notes to highlights (`Ctrl+N`) and export them to Markdown.

## Split View
- Press `Ctrl+\` to split the pane right, allowing side-by-side reading (e.g., PDF on left, Markdown note on right).
- Use the pane controls in the top-right to split down or close panes.
- Each pane maintains its own search context.

## Study Tracker
- Toggle the Study Tracker with `Ctrl+Shift+T`.
- Record active reading sessions, set project stages, and track progress without leaving your vault.
