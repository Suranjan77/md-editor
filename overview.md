# MD Editor Overview

MD Editor is a modern, lightweight Markdown editor built with CodeMirror 6 and Tauri, designed to provide a beautiful and seamless writing experience.

## Key Features

- **Live Block Previews**: Enjoy a rich, Typora-like experience where markdown syntax such as headers, blockquotes, lists, and formatting dynamically transforms into stylized text elements as you type.
- **Embedded Math Support**: Write complex algebraic equations using LaTeX syntax (`$...$` for inline, `$$...$$` for blocks). Equations render seamlessly inline using `katex`.
- **Intelligent Tooltips**: While editing a math equation, you immediately get an interactive, elevated tooltip previewing your equation exactly as it will lock in when finished.
- **Syntax Highlighting for Code Blocks**: Any code snippets you wrap in triple backticks are automatically highlighted using `highlight.js`, preserving readability natively inside the editing view without manual styling.
- **Image Previews**: Markdown image links dynamically render directly inside the editor screen, providing a seamless visual authoring workflow.
- **Interactive Checklist Widgets**: Turn simple `- [ ]` syntax into live checkbox widgets that can be interactively clicked within the editor. 

With its dark-glass elevated interface and live preview capabilities, MD Editor is completely optimized to keep you focused exclusively on content creation.
