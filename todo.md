# MD Editor

This project aims to be a lightweight, high-performance Markdown editor with features inspired by **Obsidian** and **Typora**.

## High Priority

- [x] **Find/Replace:** Find/Replace per file.
- [x] **Vault Search:** Global full-text search across all markdown files in the current vault.
- [x] **Auto hide panel:** Right now the files and folders (vault) panel is hidden and only shows up if mouse pointer is near. Instead it should always be visible and only collapsed if `Ctrl+\` instead of hiding the sidebar, whoch should always be visible.
- [x] **File Management:**
    - [x] Rename files/folders from the sidebar.
    - [x] Delete files/folders (with confirmation).
    - [x] Create folders from the sidebar.

## Editor Enhancements

- [ ] **Drag & Drop Attachments:** Drag an image into the editor to automatically copy it to an `attachments/` folder and insert the link.

## Visualisations & Navigation

- [ ] **Graph View:** A 2D interactive visualization showing how files are linked via Wikilinks.
- [ ] **Recent Files:** A "Quick Switcher" (`Ctrl+P`) to search and jump to recently opened files.
- [ ] **Backlinks Improvements:** Show a snippet of context for each backlink.

## Settings & Customization

- [ ] **Settings Panel:**
    - [ ] Toggle Line Numbers.
    - [ ] Change Editor Font Size and Family.

## Advanced & Sync

- [ ] **Git Integration:** Auto-commit changes on save or a manual "Sync" button.
