## How to use it?
- Run `npm install` then `cargo tauri build` from the root folder. You might need to install tauri cli `cargo install tauri-cli`
- To build a portable zip use `npm run package:portable` on the root folder (works on linux and windows, I don't have mac to test). Extract and double click the `md-editor.exe` file.
- Then figure out.
